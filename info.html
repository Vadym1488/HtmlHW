Tags

  • h1: обозначает самый важный заголовок (заголовок верхнего уровня), а тег h6 обозначает подзаголовок самого нижнего уровня. Буква «h» в названии тега — это первая буква английского «heading».
  • p: элемент текстового абзаца
  • br: элемент переноса строки, он используется для создания новой строки или начала нового абзаца в блоке текста. Это первая строка.
  • strong: элемент выделения важного текста
  • em: элемент выделения важного текста курсивом
  • ul: используется для неупорядоченного списка, в частности для маркированного списка.
  • ol: используется для упорядоченного списка, в частности для пронумерованного списка.
  • blockquote: используется для вставки цитат на страницу. Цитата обычно отображается с отступом от остального текста.
  • q: используется для выделения коротких цитат или комментариев.
  • pre: используется для отображения предварительно отформатированного текста. Он сохраняет все пробелы и переносы строк, а также отображает содержимое в моноширинном шрифте.
  • div: является универсальным блочным элементом и предназначен для группирования элементов документа с целью изменения вида содержимого через стили.
  • li: используется для создания элементов списка.
